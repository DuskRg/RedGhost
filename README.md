# Red Ghost

**Red Ghost — независимый игровой стартап, создающий атмосферную научно-фантастическую игру о космосе.**

## О проекте
THE LAST WAR — это проект, который объединяет технологии и творчество. Мы создаем уникальный космос.

- Лор и сценарий: **В работе**
- Поиск художников и 3D дизайнеров: **Актуально**
